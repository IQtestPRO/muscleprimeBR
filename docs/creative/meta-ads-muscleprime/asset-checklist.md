## Selos e informacoes

- [ ] Usar apenas selos reais/verificaveis.
- [ ] Nao usar "aprovado pela Anvisa" sem prova documental.
- [ ] Nao usar "original garantido" sem base documental.
- [ ] Nao usar desconto, preco, frete ou urgencia sem condicao real validada.

## Disclaimers recomendados

- [ ] "As informacoes do catalogo nao substituem orientacao profissional."
- [ ] "Produtos sujeitos a disponibilidade e regulamentacao aplicavel."

- [ ] Para campanhas sensiveis: segmentacao 18+.
